# WELLGUARD-XAI- NeuroGuard Screen Control

An Explainable AI-Based Adaptive Digital Wellbeing and Screen-Time Governance Framework System

Overview

NeuroGuard Screen Control is a research-driven Android and Web-based Digital Wellbeing platform designed to reduce excessive screen exposure and promote healthier technology habits among children, adolescents, young adults, and families.

The system combines Explainable Artificial Intelligence (XAI), adaptive screen-time governance, parental supervision, and digital wellness analytics to create a safer and more balanced digital environment.

Unlike traditional screen-time applications that only display usage statistics, NeuroGuard actively assists guardians in controlling excessive digital device usage through intelligent monitoring, adaptive restrictions, and transparent AI-generated recommendations.

---

Research Motivation

Excessive digital device usage has been associated with:

- Reduced attention span
- Sleep disturbances
- Academic performance decline
- Mental fatigue and stress
- Social isolation
- Reduced physical activity
- Digital addiction behaviors
- Cognitive overload among children and adolescents

NeuroGuard aims to provide a practical and explainable solution to these challenges.
